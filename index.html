Collatz Conjecture Tester

view it!

Numberphile's video on the Collatz Conjecture inspired me to make this. It's built using BigInteger JS to work around Javascripts relatively small limit of 253-1. There is not a limit to the size of the number that someone enters. The only limit is 10000 iterations through the Conjecture. All calculations are done client side so I felt I had to impose some limit to ensure that browsers wouldn't become indefinitely unresponsive.
